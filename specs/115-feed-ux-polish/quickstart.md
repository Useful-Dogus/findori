# Quickstart: 유저 피드 화면 UX 폴리싱

## 1. 상단 정보 위계 확인

1. `npm run dev`를 실행한다.
2. `/feed/latest` 또는 날짜 피드를 연다.
3. 첫 화면에서 카드 소비 진입부가 면책 문구보다 먼저 보이는지 확인한다.
4. 소개 문구와 사용법 문구가 제거되었는지 확인한다.

## 2. 모바일 헤더와 컨텍스트 확인

1. 브라우저를 모바일 뷰포트로 전환한다.
2. 이전 발행일/최신 피드 버튼이 치우치지 않는지 확인한다.
3. 데이터가 없는 컨텍스트 슬롯이 보이지 않는지 확인한다.

## 3. 카드 가독성과 로딩 확인

1. 카드 첫 진입과 카드 전환 시 텍스트만 먼저 튀어나오지 않는지 확인한다.
2. 밝은 배경 이미지 카드에서 텍스트 가독성이 유지되는지 확인한다.
3. 긴 한국어 문장과 출처명이 단어 중간에서 끊기지 않는지 확인한다.
4. 카드 본문과 출처가 말줄임 없이 읽히는지 확인한다.

## 4. 데스크톱 밀도 확인

1. 넓은 데스크톱 뷰포트에서 피드를 연다.
2. 카드 배경만 커져 보이지 않고 내부 수치/본문/출처가 더 읽기 좋게 확장되었는지 확인한다.
3. 오버플로로 인해 카드가 깨지지 않는지 확인한다.
