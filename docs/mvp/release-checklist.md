## 2. 핵심 기능 점검

### 2.1 발행/피드

- [ ] 장 마감 후 당일 카드 뉴스가 정상 발행된다
- [ ] 세로 스크롤(그룹)과 가로 스와이프(카드)가 의도대로 동작한다
- [ ] 카드 진행 상태(1/N)가 정확히 표시된다
- [ ] 핵심 카드에 변동 사실(수치/방향)과 변동 이유가 함께 노출된다
- [ ] 코스피/나스닥/USD/KRW 맥락 카드가 당일 피드에 노출된다

### 2.2 공유/비로그인

- [ ] 이슈 permalink가 생성된다
- [ ] 공유 링크가 비로그인 상태에서 열람된다
- [ ] 링크 미리보기(썸네일/제목/요약)가 정상 노출된다

### 2.3 출처/고지

- [ ] 카드별 출처 링크가 누락 없이 포함된다
- [ ] 출처 링크가 실제로 열리는지 확인했다
- [ ] 투자 자문 오해 방지 문구가 피드/공유 랜딩에 노출된다

---

## 3. 웹 정합성/품질 점검

- [ ] 모바일에서 가독성(폰트/간격/터치 영역)이 확보된다
- [ ] 데스크톱에서 핵심 열람 동선이 유지된다
- [ ] 로딩/빈 화면/오류 화면이 각각 준비되어 있다
- [ ] 공유 링크 최초 진입 시 체감 지연이 과도하지 않다

---

## 4. 계측/운영 점검

- [ ] 핵심 이벤트 로그가 수집된다 (`feed_opened`, `card_swiped`, `share_clicked`, `source_clicked`)
- [ ] 오류 로그 수집/조회 경로가 준비되어 있다
- [ ] 운영 문의/이슈 대응 담당자(본인) 확인 완료

---

## 5. 2인 파일럿 점검

대상:

- [ ] 모바일 선호 사용자 1명
- [ ] 데스크톱 선호 사용자 1명

검증 질문:

- [ ] 첫 화면에서 오늘 핵심 이슈를 10초 내 파악했는가
- [ ] 특정 종목의 가격 변동 이유를 카드만 보고 설명할 수 있는가
- [ ] 코스피/나스닥/USD-KRW 변동 이유를 카드만 보고 설명할 수 있는가
- [ ] 필요할 때 출처 확인이 어렵지 않았는가
- [ ] 공유받은 링크에서 로그인 없이 맥락 이해가 가능했는가

결과 처리:

- [ ] 피드백 정리 및 개선 백로그 등록 완료

---

## 6. Go/No-Go

Go 조건:

- [ ] 핵심 기능 블로커 없음
- [ ] 정책 위반 리스크 없음
- [ ] 2인 파일럿에서 핵심 이해 목표 충족

No-Go 조건:

- [ ] 공유 링크 비로그인 열람 실패
- [ ] 출처 누락/고지 누락
- [ ] 핵심 변동 이유 카드 또는 지수/환율 맥락 카드 누락
- [ ] 모바일/데스크톱 중 한 환경에서 열람 동선 붕괴
